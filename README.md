# 🚀 Mini CRM - Lead Management System

A full-stack Lead Management Dashboard built as part of the **Future Interns** Internship (Task 2). This application allows administrators to securely manage, track, and modify lead data in real-time.

## 🌟 Features
- **Secure Admin Access:** Password-protected dashboard gate.
- **Real-time Database:** Powered by **Supabase (PostgreSQL)**.
- **Full CRUD Functionality:** - **Create:** Add new leads via a custom form.
  - **Read:** Fetch and display lead data in a dynamic table.
  - **Delete:** Remove leads instantly from the database.
- **Live Counter:** Automatically tracks the total number of leads stored.
- **Responsive UI:** Clean, professional interface with status indicators.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend-as-a-Service:** Supabase API
- **Database:** PostgreSQL
- **Hosting:** GitHub Pages
